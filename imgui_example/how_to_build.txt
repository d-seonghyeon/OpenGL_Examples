2. 터미널에서
nano ~/.zprofile -> export PATH="/opt/homebrew/bin:$PATH" ->source ~/.zprofile
저장: Ctrl + 0

3. cmd +shift +P -> preferences user settings (json) -"cmake.cmakePath": "/opt/homebrew/bin/cmake" 추가
